Исключения находятся в методе главного класса StartingPoint - public void run().
Один Exception сработает в случае прерывания потока и выдаст информацию об этой ошибке. 
Другой сработает, если вдруг почему-то лиса не прорисуется. Кастомный exception описан в классе CastomException. Выбрасывается он в классе Fox в методе update(). А обрабатывается в классе StartingPoint - public void run()
